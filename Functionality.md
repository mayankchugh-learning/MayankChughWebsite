# Application Overview
This is a full-stack portfolio application for Mayank Chugh, focusing on his expertise as a Generative AI Engineer and Solutions Architect.

## Core Functionality
1. **Dynamic Portfolio**: Showcases professional experience, projects, and skills fetched from a PostgreSQL database.
2. **AI Assistant**: A floating chat agent powered by OpenAI (gpt-5) that answers questions about Mayank's career.
3. **Contact Management**: A form for visitors to send messages directly to the database.
4. **Admin Data Management**: Seeded with real-world professional data for immediate deployment.

## Technical Stack
- **Frontend**: React with TypeScript, Tailwind CSS, Framer Motion, and shadcn/ui.
- **Backend**: Node.js with Express, Drizzle ORM for database management.
- **Database**: PostgreSQL.
- **AI**: OpenAI Integration via Replit AI Integrations.
