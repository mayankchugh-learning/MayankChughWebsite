# AI Agent Generation Prompts

To generate a similar application in the future, use the following sequence of prompts:

## 1. Initial Architecture & Schema
"Create a full-stack portfolio application using React, Express, and PostgreSQL. Define a schema in Drizzle for Profile, Experience, Projects (with tech stack and links), and Skills (with proficiency levels). Include a table for visitor messages."

## 2. Professional Content Seeding
"Seed the database with professional data for a [Job Title]. Ensure the bio highlights [Key Skills] and includes social links for LinkedIn, GitHub, and YouTube. Set up an avatar image placeholder."

## 3. AI Agent Integration
"Add a floating AI chat assistant to the portfolio. It should be powered by OpenAI and trained on the user's professional profile. Ensure it asks for the visitor's email and persists session history. Use a professional and helpful personality."

## 4. UI/UX Polishing
"Design the UI with a modern, dark-themed aesthetic using Tailwind CSS and shadcn/ui. Add scroll animations with Framer Motion and ensure a responsive layout for mobile and desktop."
